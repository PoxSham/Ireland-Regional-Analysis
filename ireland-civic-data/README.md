# ireland-civic-data

Data pipeline for the Ireland Civic Data Project.

## Structure

```
ireland-civic-data/
  schema/
    geography.json   — Canonical NUTS2/NUTS3/county lookup + join rules
    measures.json    — Canonical measure definitions for all 9 MVP pillars
  etl/
    fetch_cso.js     — CSO PxStat API ETL (12 datasets)
    fetch_eurostat.js — Eurostat REST API ETL (6 datasets)
    fetch_ogp.js     — OGP procurement CSV ETL (eTenders + quarterly awards)
    run_all.js       — Orchestrator + QA checks
  data/
    processed/       — Normalised JSON outputs + manifests (tracked)
    raw/             — Raw API responses (gitignored — regenerated by ETL)
    snapshots/       — Frozen builds for published releases
  docs/
    sipo_extraction_plan.md — Plan for SIPO lobbying data extraction (Phase 2)
```

## Running ETL

```bash
node etl/run_all.js           # All sources
node etl/run_all.js --cso     # CSO only
node etl/run_all.js --eurostat # Eurostat only
node etl/run_all.js --ogp     # OGP procurement only
node etl/run_all.js --qa-only # QA checks without re-fetching
```

## Sources

| Source | Datasets | Pillars |
|--------|----------|---------|
| CSO PxStat | CIRGDP01, CIRGDP03, NA001, NA002, UA19, UA42, GFQ01, GFA01, GFQ03, GFA03, GFQ04, GFA04 | Regions, Ireland on Paper, Public Spending, Taxes & Revenue |
| Eurostat | prc_ppp_ind, ilc_di03, ilc_li02, hlth_sha11_hf, hlth_sha11_hc, nama_10r_3gdp | Ireland vs EU, Healthcare, Ireland on Paper |
| OGP / eTenders | Quarterly awards 2023–2024, eTenders bulk (~100k notices) | Procurement & Contracts |
| SIPO lobbying.ie | Phase 2 — HTML extraction planned | Lobbying & Influence |
